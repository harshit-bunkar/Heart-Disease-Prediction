## 📌 Project Overview

To build a model that predicts whether a person has heart disease based on their medical attributes, supporting early diagnosis.

## 🎯 Objectives

- Analyze and understand the dataset
- Clean and preprocess the data
- Build a model using **Logistic Regression**
- Train the model and make predictions
- Evaluate the model's performance

## 📊 Dataset Information

A dataset of 303 patient records with 13 medical features and a target indicating the presence of heart disease.

**Key features:**

- Age & Sex
- Chest Pain Type
- Resting Blood Pressure
- Serum Cholesterol
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression (oldpeak) & Slope
- Number of Major Vessels
- Thalassemia

**Target: 0 = Healthy, 1 = Has Heart Disease**

## 🤖 Model

**Algorithm used:** Logistic Regression

## 📈 Results

| Metric | Value |
|--------|-------|
| Test Accuracy | **82.0%** |
| Training Accuracy | **85.1%** |

A Logistic Regression model was trained on the patient data. It achieved 85.1% accuracy on training data and 82.0% accuracy on unseen test data, showing it can support doctors in early heart-disease risk screening.
